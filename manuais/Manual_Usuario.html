Manual do Usuario

Sistema: Dimensionamento de Enfermagem TASY

1. Primeiro acesso

  1. Acesse login.html.
  2. Digite usuario e senha.
  3. Se ainda nao tiver acesso, clique em Solicitar cadastro.

2. Solicitar cadastro

  1. Preencha Nome, Usuario, Senha, Perfil solicitado e Setores.
  2. Clique em Enviar solicitacao.
  3. Aguarde aprovacao do Administrador (T33).

3. Uso da Tela Principal

  • Selecione o Filtro por setor.
  • Preencha dados da Tabela Principal (8 tecnicos).
  • Registre Controle Diario e acompanhe graficos.
  • Atualize SCP para calculo de tempo assistencial.

4. Botoes da Linha

  • Confirmar: bloqueia a linha para evitar alteracao acidental.
  • Reiniciar: desbloqueia para nova edicao.
  • Remover: limpa os dados da linha.

5. Exportacao

  • Use Exportar CSV para compartilhar informacoes.
  • Use Exportar PDF via impressao do navegador.

6. Acesso por celular e PC

  • Use o mesmo link do sistema em qualquer dispositivo autorizado.
  • Se disponivel, instale como app (PWA) pelo navegador.

7. Regras importantes

  • Nao compartilhe senha.
  • Preencha dados reais do plantao.
  • Em caso de erro de acesso, contate o ADM.
