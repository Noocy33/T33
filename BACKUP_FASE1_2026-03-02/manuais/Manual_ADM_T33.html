Manual do Administrador (T33)

Sistema: Dimensionamento de Enfermagem TASY

Credencial ADM: Usuario T33 | Senha 123456

Usuario operacional padrao: TESTET33 | Senha 123456

1. Acesso e modulos

  1. Acesse login.html e entre com o usuario ADM.
  2. Modulos principais: Dimensionamento, Grafico Equipe, Escala de Tecnico de Enfermagem, Controle Diario e SCP.
  3. Gestao de Acesso permanece exclusiva para o ADM T33.

2. Gestao de usuarios

  1. Em Gestao de Acesso, use os campos Nome, Usuario, Senha, Perfil e Setores.
  2. Use Salvar usuario para criar/atualizar.
  3. Use Editar para carregar usuario na tela e ajustar dados.
  4. A conta T33 e protegida contra remocao.

3. Solicitacoes de cadastro

  • Na tela de login, o link Solicitar cadastro esta ativo para PC e celular.
  • As solicitacoes entram na tabela Solicitacoes pendentes.
  • Use Aprovar ou Reprovar conforme validacao.

4. Avatar e fotos

  • Login/splash/topo usam avatar padrao quando nao ha foto personalizada.
  • A foto do topo e visivel para T33 e TESTET33.
  • Upload da foto do topo liberado somente para T33.
  • Na Escala de Tecnico de Enfermagem, os 8 tecnicos usam avatar fixo (sem troca).

5. Regras operacionais

  • Confirmar bloqueia a linha.
  • Reiniciar desbloqueia para edicao.
  • Remover limpa dados da linha.
  • Setores UTI aceitam leitos com 4 digitos; demais setores, 3 digitos.

6. Abertura, cache e logout

  • Splash configurada para 3 segundos.
  • Botao Sair redireciona para login.html?nocache=1.
  • Versoes de assets e Service Worker sao atualizadas para reduzir cache antigo.
